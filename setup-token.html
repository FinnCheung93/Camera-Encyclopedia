GitHub Token 本机设置

不要把 Personal Access Token 写进 index.html、config.js 等会上传到 GitHub 的文件，否则会触发 Secret scanning 并被拦截。请只在下面粘贴一次，保存后 Token 仅存在于当前浏览器本机（localStorage）。

仓库、分支仍由 config.js 中的 owner / repo / branch 决定；此处只负责 Token。

给别人看前台：把 GitHub 仓库设为 Public 即可，对方不用配置 Token。 只有你要用「后台」改数据并写回 GitHub 时，才需要在本页保存 PAT。
